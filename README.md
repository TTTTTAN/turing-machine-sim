### 四位加法器 4-bit-adder

可以进行四位二进制数的加法。和可以高至五位。

被加数在加号左侧，加数在加号右侧，结果显示在加号左侧。详见`description.txt`

#### e1.

`TAPE INPUT: 0010+0101B`

`TAPE OUTPUT: 0111+aaaaB`

#### e2.

`TAPE INPUT: 01111+0001B`

`TAPE OUTPUT: 10000+aaaaB`
